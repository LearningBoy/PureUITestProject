Google VR for Unity
=
##### 以Unity为开发工具，配合gvr-unity-sdk，开发运行于Android平台、适用于CardBoad等头戴式VR的小程序

****

使用教程
=
#####1、	下载工程到本地目录，并通过Unity(Version 5.4.1f1)打开
#####2、	在工程目录窗口，通过Assets->_Scene目录,双击打开MainUI
#####3、 运行，长按“alt”键，移动鼠标，改变焦点方向，测试相关功能

****

功能效果
=
#####1、 获得焦点后的单击事件 ---- 已实现
#####2、 获得焦点后以进度条的形式进行倒计时 ---- 已实现
